### installation
```
npm i @pritish__mishraa/file_system_organizer
```

### run command
``` 
orgo 
```
#### options
> ``` orgo help ```
> shows all the command 

> ``` orgo organize ```
> creates the organized folder where it copys and seperated the file
> according to their type

> ``` orgo tree ```
> displays the entire folder in tree structure
